## Getting Started

Clone the repo on your local, you will need to have Node JS installed to be able to run the project.

Questions? Comments?
jr.torsa@gmail.com

### Prerequisites

1. Node installed

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jrtorsa/mrspex.git
   ```
2. Install NPM packages
   ```sh
   cd mrspex
   npm install
   or
   yarn
   ```
3. Run the client project
   ```sh
    npm start
    or
    yarn start
   ```
4. open your browser
   http://localhost:3000
